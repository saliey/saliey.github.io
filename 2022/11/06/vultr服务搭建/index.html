vultr服务搭建

官网购买服务
https://www.vultr.com/

ssh 进入服务器：
.# apt-get install redir
.# sudo iptables -A INPUT -p tcp –dport 443 -j ACCEPT
.# sudo redir :443 1.1.1.1:443

然后在 vultr 的后台打开防火墙的端口：
Firewall -> new -> 打开对应端口

然后远端就实现了端口数据转发；

yum 内安装 redir

不过这个是老版本，应该下载源码，然后自编译安装

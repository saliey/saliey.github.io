github个人博客搭建

1. 得先确认自己的 git 项目能正常拉取和提交

采用 ssh 的方式 clone
git clone git@github.com:saliey/ctest.git
vim readme.txt
git add ..
git commit ..
git push

2. 需要配置 ssh key

这里需要本地配置 ssh key
ssh-keygen -t rsa -C “sss@email.com“

将 id_rsa.pub 的内容拷贝到 github 的后台；

3. hexo 的搭建

mkdir newFloder
npm install -g hexo-cli
npm install -g hexo
hexo init
npm install
hexo g
hexo s 这里就可以验证是否安装成功

然后配置 _config.yml 文件
deploy:
type: ‘git’
repo: git@github.com:saliey/saliey.github.io.git
branch: main

然后 npm install hexo-deployer-git –save

最后
hexo g
hexo d 部署

4. 干扰的事情

后台的 token 未设置限制时间；token 是 xxx

且 ssh 已经添加了三个，对应本季 .ssh/id_rsa.pub .ssh/pub_github.pub 和一个已经删除的；

5. hexo 发布文章

hexo new test
然后编写文章，
hexo g
hexo d 就发布了；

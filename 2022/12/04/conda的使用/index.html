conda的使用

Posted on 2022-12-04

清华源：
https://mirrors.tuna.tsinghua.edu.cn/help/anaconda/

创建环境：
conda create –name py310 python=3.10

切换环境：
conda activate py310

退出环境：
conda deactivate

删除环境：
conda remove -n py310 –all
conda env remove –name main

安装环境包：
conda install requests

显示当前已安装包的列表：
conda list

将当前已经安装的软件包导出来：
conda list –explicit > test1.txt
安装导出的安装包的列表：
conda install –file ./test.txt

添加源：
conda config –add channels https://mirrors.ustc.edu.cn/anaconda/pkgs/free/
conda config –add channels https://mirrors.ustc.edu.cn/anaconda/cloud/bioconda/osx-64/
conda config –add channels https://mirrors.ustc.edu.cn/anaconda/cloud/bioconda/

conda config –show
